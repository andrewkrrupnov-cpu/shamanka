# assets/cards/ — изображения колоды

78 файлов, имя = индекс карты в `FULL_DECK` (см. `src/deck.py`):

- `00` — Шут, `01` — Маг, … `21` — Мир (старшие арканы, порядок из spreads_source.md)
- `22` — Туз Кубков, … `77` — Король Жезлов (младшие: масти Кубки, Пентакли, Мечи, Жезлы)

Расширение — любое из `jpg / jpeg / png / webp`. Код (`src/cards.py`) находит файл
по индексу и расширению сам.

Сейчас в репозитории лежит настоящая колода (78 PNG). Чтобы заменить на другую —
положи файлы с теми же именами (`00.png`…`77.png`), бот подхватит без правок кода.

Для быстрых заглушек (если колоды нет) есть генератор:

```
pip install pillow
python scripts/gen_placeholder_cards.py
```
